1) What is a Doubly Linked List?

A Doubly Linked List (DLL) is a linked list where every node has three parts:

prev

Pointer to the previous node.

data

The actual value stored inside the node.

next

Pointer to the next node.

Because every node has prev and next, we can move forward from head to tail and backward from tail to head.

2) Important Pointers

PointerMeaning
headPoints to the first node.
tailPoints to the last node.
nullptr / NULLMeans the pointer points to nothing.

4) Operations Explained

Append(data)

Adds a new node at the end. If list is empty, head and tail become the new node. Otherwise, link old tail to new node, then update tail.

Insert(data, index)

Adds a new node before the node currently at that index. It must update four links: new prev, new next, previous next, and current prev.

Delete(data)

Finds the node by value, then reconnects its previous and next nodes so the deleted node is skipped.

Search(data)

Starts at head and moves using next until it finds the value or reaches NULL.

Display()

Traverses from head to tail using next.

displayFromEnd()

Traverses from tail to head using prev.

5) C++ Node and Class Structure

struct DLLNode
{
    int Data;              // value stored in the node
    DLLNode* nextNode;     // points to the next node
    DLLNode* prevNode;     // points to the previous node
};

class DoublyLL
{
private:
    DLLNode* head;         // first node
    DLLNode* tail;         // last node
public:
    DoublyLL();
    void Append(int data);
    void Insert(int data, int index);
    void Delete(int data);
    bool Search(int data);
    void Display();
    void displayFromEnd();
    int Size();
};

6) Insert Order: Very Important

When inserting in the middle, do not lose the old links. Correct order:

newNode->prevNode = current->prevNode; // new points backward
newNode->nextNode = current;           // new points forward
current->prevNode->nextNode = newNode; // previous points to new
current->prevNode = newNode;           // current points back to new

7) Singly vs Doubly Linked List

Singly Linked ListDoubly Linked List
Each node has data + next.Each node has prev + data + next.
Moves forward only.Moves forward and backward.
Uses less memory.Uses more memory because of prev pointer.
Harder to reverse traverse.Reverse traversal is easy using tail and prev.

8) Common Exam / Discussion Questions

Q: Why DLL uses more memory?
Because each node stores an extra pointer called prev.

Q: Why is deletion easier in DLL?
Because the node already knows its previous node using prev.

Q: What happens if head is NULL?
The list is empty.

Q: Why do we need tail?
To append faster and traverse from the end.

1. Function: Count nodes in a Doubly Linked List

Task: Write a C++ class method int DoublyLL::Size() that returns the number of nodes in the list.
int DoublyLL::Size()
{
    int counter = 0; // Create a counter to count how many nodes exist.
    DLLNode* temp = head; // Start traversal from the first node.
    while (temp != nullptr) // Keep moving while we are still inside the list.
    {
        counter++; // Count the current node.
        temp = temp->nextNode; // Move to the next node.
    }
    return counter; // Return the total number of nodes.
}

2. Function: Display nodes from head to tail

Task: Write a C++ class method void DoublyLL::Display() that prints all node values from the beginning to the end.
void DoublyLL::Display()
{
    DLLNode* temp = head; // Start from the first node in the list.
    while (temp != nullptr) // Continue until we pass the last node.
    {
        cout << temp->Data << " "; // Print the value stored in the current node.
        temp = temp->nextNode; // Move forward using the next pointer.
    }
}

3. Function: Display nodes from tail to head

Task: Write a C++ class method void DoublyLL::displayFromEnd() that prints all values in reverse order.
void DoublyLL::displayFromEnd()
{
    DLLNode* temp = tail; // Start from the last node in the list.
    while (temp != nullptr) // Continue until we move before the first node.
    {
        cout << temp->Data << " "; // Print the value of the current node.
        temp = temp->prevNode; // Move backward using the previous pointer.
    }
}

4. Function: Search for a value

Task: Write a C++ class method bool DoublyLL::Search(int data) that returns true if the value exists, otherwise false.
bool DoublyLL::Search(int data)
{
    DLLNode* temp = head; // Start searching from the first node.
    while (temp != nullptr) // Keep searching until the list ends.
    {
        if (temp->Data == data) // Check if the current node has the target value.
        {
            return true; // Value found, so return true immediately.
        }
        temp = temp->nextNode; // Move to the next node.
    }
    return false; // The loop ended, so the value was not found.
}

5. Function: Append a new node at the end

Task: Write a C++ class method void DoublyLL::Append(int data) that adds a new node after the current tail.
void DoublyLL::Append(int data)
{
    DLLNode* newNode = new DLLNode; // Create a new node in dynamic memory.
    newNode->Data = data; // Store the input value in the new node.
    newNode->nextNode = nullptr; // The new node will be last, so its next is null.
    newNode->prevNode = tail; // Its previous node is the old tail.
    if (head == nullptr) // Check if the list is empty.
    {
        head = newNode; // In an empty list, head points to the new node.
        tail = newNode; // In an empty list, tail also points to the new node.
        return; // Stop because insertion is complete.
    }
    tail->nextNode = newNode; // Link the old tail to the new node.
    tail = newNode; // Update tail to become the new node.
}

6. Function: Insert at the head

Task: Write a C++ class method void DoublyLL::InsertHead(int data) that inserts a new node before the current head.
void DoublyLL::InsertHead(int data)
{
    DLLNode* newNode = new DLLNode; // Create a new node.
    newNode->Data = data; // Put the input value inside the node.
    newNode->prevNode = nullptr; // New head has no previous node.
    newNode->nextNode = head; // New node points forward to the old head.
    if (head == nullptr) // If the list is empty, there is no old head.
    {
        head = newNode; // Head becomes the new node.
        tail = newNode; // Tail also becomes the new node.
        return; // Stop after handling the empty list case.
    }
    head->prevNode = newNode; // Old head points backward to the new node.
    head = newNode; // Update head to the new node.
}

7. Function: Delete the first occurrence of a value

Task: Write a C++ class method void DoublyLL::Delete(int data) that deletes the first node containing the given value.
void DoublyLL::Delete(int data)
{
    DLLNode* temp = head; // Start from the head to search for the value.
    while (temp != nullptr && temp->Data != data) // Move until value is found or list ends.
    {
        temp = temp->nextNode; // Go to the next node.
    }
    if (temp == nullptr) // If temp is null, the value was not found.
    {
        return; // Nothing to delete.
    }
    if (temp == head) // If deleting the first node.
    {
        head = head->nextNode; // Move head to the second node.
    }
    if (temp == tail) // If deleting the last node.
    {
        tail = tail->prevNode; // Move tail to the previous node.
    }
    if (temp->prevNode != nullptr) // If there is a node before temp.
    {
        temp->prevNode->nextNode = temp->nextNode; // Link previous node to next node.
    }
    if (temp->nextNode != nullptr) // If there is a node after temp.
    {
        temp->nextNode->prevNode = temp->prevNode; // Link next node back to previous node.
    }
    delete temp; // Free the deleted node from memory.
}

8. Function: Insert before a given index

Task: Write a C++ class method void DoublyLL::Insert(int data, int index) that inserts a new node before the node currently at the given index.
void DoublyLL::Insert(int data, int index)
{
    if (index < 0 || index > Size()) // Check if the index is invalid.
    {
        cout << "Index out of bounds!" << endl; // Print an error message.
        return; // Stop the function.
    }
    if (index == 0) // If inserting at the beginning.
    {
        InsertHead(data); // Reuse the InsertHead function.
        return; // Stop because insertion is complete.
    }
    if (index == Size()) // If inserting after the current last node.
    {
        Append(data); // Reuse the Append function.
        return; // Stop because insertion is complete.
    }
    DLLNode* current = head; // Start from head to reach the required index.
    for (int i = 0; i < index; i++) // Move index times.
    {
        current = current->nextNode; // Move to the next node.
    }
    DLLNode* newNode = new DLLNode; // Create the new node.
    newNode->Data = data; // Store the input value.
    newNode->prevNode = current->prevNode; // New node points back to the node before current.
    newNode->nextNode = current; // New node points forward to current.
    current->prevNode->nextNode = newNode; // Previous node now points forward to new node.
    current->prevNode = newNode; // Current node now points backward to new node.
}

9. Function: Delete all even values

Task: Write a C++ class method void DoublyLL::DeleteEven() that removes every node whose value is even.
void DoublyLL::DeleteEven()
{
    DLLNode* temp = head; // Start checking from the first node.
    while (temp != nullptr) // Continue until all nodes are checked.
    {
        DLLNode* next = temp->nextNode; // Save the next node before deleting anything.
        if (temp->Data % 2 == 0) // Check if the current value is even.
        {
            if (temp == head) // If the current node is the head.
            {
                head = temp->nextNode; // Move head to the next node.
            }
            if (temp == tail) // If the current node is the tail.
            {
                tail = temp->prevNode; // Move tail to the previous node.
            }
            if (temp->prevNode != nullptr) // If there is a previous node.
            {
                temp->prevNode->nextNode = temp->nextNode; // Skip temp from the forward direction.
            }
            if (temp->nextNode != nullptr) // If there is a next node.
            {
                temp->nextNode->prevNode = temp->prevNode; // Skip temp from the backward direction.
            }
            delete temp; // Delete the even node from memory.
        }
        temp = next; // Continue traversal using the saved next pointer.
    }
}

10. Function: Insert using tail traversal

Task: Write a C++ class method void DoublyLL::InsertReverse(int data, int index) that starts from tail and inserts a node before the node at the given index.
void DoublyLL::InsertReverse(int data, int index)
{
    int size = Size(); // Store the list size to avoid recalculating it many times.
    if (index < 0 || index > size) // Check for an invalid index.
    {
        cout << "Index out of bounds!" << endl; // Print an error message.
        return; // Stop the function.
    }
    if (index == 0) // If insertion is at the beginning.
    {
        InsertHead(data); // Insert before the current head.
        return; // Stop because insertion is complete.
    }
    if (index == size) // If insertion is at the end.
    {
        Append(data); // Add after the current tail.
        return; // Stop because insertion is complete.
    }
    DLLNode* current = tail; // Start traversal from the tail.
    int counter = size - 1; // Tail is located at index size - 1.
    while (counter > index) // Move backward until reaching the wanted index.
    {
        current = current->prevNode; // Move one step toward the head.
        counter--; // Decrease the index counter.
    }
    DLLNode* newNode = new DLLNode; // Create the node to insert.
    newNode->Data = data; // Store the input value.
    newNode->prevNode = current->prevNode; // New node points back to current previous node.
    newNode->nextNode = current; // New node points forward to current node.
    current->prevNode->nextNode = newNode; // Previous node links forward to new node.
    current->prevNode = newNode; // Current node links backward to new node.
}
